FAISS и поиск ближайших соседей
FAISS (Facebook AI Similarity Search) — это библиотека с открытым исходным кодом для эффективного поиска ближайших соседей в многомерных пространствах. Она разработана исследовательской группой Facebook AI Research. FAISS поддерживает различные типы индексов, оптимизированных под разные задачи и объёмы данных. Простейший индекс — IndexFlatIP — выполняет точный поиск по скалярному произведению. Для нормированных векторов скалярное произведение эквивалентно косинусному сходству. IndexFlatL2 использует евклидово расстояние. FAISS также предоставляет приближённые методы поиска, такие как IVF (Inverted File Index) и HNSW (Hierarchical Navigable Small World). Эти методы значительно ускоряют поиск на больших наборах данных за счёт небольшой потери точности. FAISS позволяет работать с векторами, которые не помещаются в оперативную память, используя mmap. Библиотека имеет Python-интерфейс и легко интегрируется с NumPy. Для поиска top-k векторов достаточно вызвать метод search, передав матрицу векторов запросов. FAISS поддерживает добавление новых векторов в существующий индекс (метод add). Однако для некоторых типов индексов (например, требующих предварительного обучения) это может быть невозможно без перестроения. В контексте RAG FAISS часто используется в качестве векторного индекса для чанков документов. Скорость поиска в FAISS может быть на порядки выше, чем прямое вычисление попарных сходств. Это делает FAISS стандартом де-факто для production-систем семантического поиска. Знание основ работы с FAISS необходимо для любого инженера, создающего retrieval-компонент.
